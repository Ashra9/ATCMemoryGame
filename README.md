# ATC Memory Game  ![Static Badge](https://img.shields.io/badge/build%20version-1.1.0-blue)


A game to test your memory on ATC airline and airport codes and callsigns!
## Latest
- Randomizer changed to a weighted randomizer to reduce multiple repeats of the same question
- Colour -> Night mode
- More location answer combinations accepted eg. "Bangkok Don Mueang International", "Bangkok" or "Don Mueang" 

##
levenshtein function from [Gustaf Andersson](https://github.com/gustf/js-levenshtein).
